 MVC Usuarios

Este proyecto es una aplicación en Python que implementa el patrón MVC (Modelo - Vista - Controlador).

Funcionalidades

- Registro de usuarios
- Validación de datos
- Login de usuarios
- Listado de usuarios

 Estructura

- modelo.py → Manejo de datos
- controlador.py → Lógica del sistema
- vista.py → Interfaz con el usuario
- main.py → Archivo principal
